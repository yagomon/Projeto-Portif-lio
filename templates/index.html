Super Batata: Jogo de Ficção Interativa

Super Batata é um jogo de terminal, desenvolvido em Python, onde uma batata orgânica tenta salvar a horta dos legumes arruaceiros que querem utilizar agrotóxicos. No game o usuário tem que tomar decisões que interferem nas possibilidades do jogo, elas podem leva-lo a derrotar todos os vilões ou ao famigerado Gameover. O projeto foi um requisito para conclusão do módulo de lógica de programação da Blue Edtech. Clique em veja mais para ter acesso ao código do projeto.

Site Evoluções do Agumon - Projeto Jogo do humor

Projeto desenvolvido para consolidar meus conhecimentos de HTML, CSS e JS. Consistiu na construção de um site estático para mostrar uma troca de informações em tela com o apertar de um botão. A proposta original era realizar a troca de humor de um personagem. Entretanto, para ambito de treino, decidi efetuar as trocas de acordo com as etapas evolutivas do personagem principal do anime Digimon, o Agumon. Clique em veja mais para acessar o site, ou clique aqui para acessar o projeto no github.

Portifólio Web

O portifólio é uma aplicação Web desenvolvida para expor os projetos que realizei até o momento. Este trabalho consistiu na construção de uma aplicação web utilizando HTML, CSS, JS e Python com Flask framework, para envio e validação do formulário de contato. Ao final do desenvolvimento do site foi realizada a conversão do mesmo para aplicação Flask e realizado deploy da aplicação no Heroku. Você já está navegando no portifólio neste momento, clique em veja mais para acessar os códigos do projeto no github.
